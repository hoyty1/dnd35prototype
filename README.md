# D&D 3.5 Hex RPG Prototype

A Unity 6.4 C# project implementing D&D 3.5 edition combat rules.

## Features

- **D&D 3.5 Combat System**: Full implementation of attack rolls, damage, critical hits
- **Character Creation**: 5-step character creation UI with all PHB races
- **Feat System**: Power Attack, Rapid Shot, Point Blank Shot, and more
- **Action Economy**: Standard, move, full-round, and swift actions
- **Tactical Combat**: Flanking, sneak attacks, dual wielding
- **Equipment System**: All PHB weapons, armor, and shields with proper stats
- **Range & Movement**: D&D 3.5 range increments and diagonal movement rules

## Getting Started

### Clone the Repository
```bash
git clone https://github.com/hoyty1/dnd35prototype.git
cd dnd35prototype
```

### Open in Unity
1. Open Unity Hub
2. Click "Add" or "Open"
3. Navigate to the `dnd35prototype` folder
4. Unity will detect the project and import assets

### Getting Updates
```bash
cd dnd35prototype
git pull
```

Unity will automatically detect changes and reload assets.
